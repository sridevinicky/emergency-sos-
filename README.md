# Emergency-sos-
Emergency SOS is a platform that helps people quickly report emergencies with a single button. It allows guests or staff to send an alert instantly so that help can arrive faster. The system improves safety by enabling quick communication and faster response during situations like medical emergencies, fires, or security issues. 🚨
# Google AI Usage
Gemini through Antigravity
In our project, AI is integrated using the Antigravity platform, which allows us to create AI-powered workflows through APIs. When a user sends an emergency message through the Rapid Crisis SOS web app, the message is sent to the Antigravity AI workflow, where it is analyzed using natural language processing. The AI identifies the type of emergency (such as fire, medical, or security) and determines the urgency level. This processed information is then sent back to the system and displayed on the staff dashboard, helping responders quickly understand the situation and take appropriate action.
